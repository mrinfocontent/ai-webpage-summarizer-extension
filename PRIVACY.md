# Privacy Policy

AI Webpage Summarizer respects user privacy.

## Data Collection
This extension does **not** collect, track, or store any personal data.

## API Keys
- API keys (Groq, OpenAI, Gemini) are provided by the user.
- Keys are stored **locally on the user's device** using Chrome storage.
- API keys are never transmitted to the developer.

## Webpage Content
- Webpage content is accessed **only when the user clicks** the Summarize button.
- Content is sent directly to the selected AI provider (Groq, OpenAI, or Gemini)
  solely for the purpose of generating a summary or answering a question.

## Third-Party Services
This extension communicates directly with:
- Groq API
- OpenAI API
- Google Gemini API

No data is shared with any other third parties.

## Data Retention
The developer does not store, log, or retain any user data.

## Changes
This privacy policy may be updated if the extension functionality changes.

## Contact
If you have questions about this privacy policy, please open an issue on the GitHub repository.
